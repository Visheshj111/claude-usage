Privacy Policy

How your data is handled by Claude Usage Tracker.

Claude Usage Tracker tracks your Claude.ai usage entirely in your browser.

What We Collect

  • Message counts and token usage visible on the page
  • Conversation titles and message counts
  • Session timing (start/stop/duration)

Where It's Stored

  • All data is stored locally in your browser's chrome.storage API
  • When you use the "Export" feature, a Markdown file is generated — nothing is transmitted
  • No data is sent to any external server or third party

What We Do NOT Collect

  • We do not read the content of your messages
  • We do not send data to any external server
  • We do not use analytics, telemetry, or tracking
  • We do not sell, share, or transmit your data

Your Control

  • Delete all tracked data anytime with the Delete all data button above
  • Uninstall the extension to remove all stored data
  • Review this policy anytime in Settings

This extension reads the DOM of claude.ai pages to detect conversation activity. All processing is done locally in your browser. Nothing is transmitted externally.
